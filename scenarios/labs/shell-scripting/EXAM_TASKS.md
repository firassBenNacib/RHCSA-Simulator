# Shell Scripting Basics - Exam Tasks
Scenario ID: shell-scripting
Mode: Exam
Time limit: 45 minutes
Objectives: shell-scripting

Practice RHCSA v9 shell scripting with loops, conditionals, command substitution, and executable scripts.

## Task 01 - User Creation Script (clientvm) - 15 pts
Create /usr/local/bin/rhcsa-user-summary to process users.csv and create missing users.

## Task 02 - User Summary Output (clientvm) - 10 pts
Write a creation/existing summary to /root/user-summary.txt.

## Task 03 - Service Check Script (clientvm) - 15 pts
Create /usr/local/bin/rhcsa-service-check to report service states from services.txt into /root/service-status.txt.

## Task 04 - Execute The Scripts (clientvm) - 10 pts
Execute both scripts successfully.
